Getting Started Guide

This guide will walk you through the installation process and prerequisites to get your admin dashboard up and running smoothly.

  • Prerequisites
    • Before you begin, make sure you have the following prerequisites installed on your system:
  • Node.js( v21.4.0 ):
    • Ensure that you have Node.js installed. You can download it fromnodejs.org.
  • Npm:
    • Don't worry npm is installed by default with Node.

Installation Steps

Follow these simple steps to install and set up Fadmin Admin Template:

  • Step 1: Unzip
    Unzip The Downloaded Folder and open it in vscode or any other code editor you want to use.

  • Step 2: Navigate to the Project Directory
    Run the command cd ./Admin in the Terminal

  • Step 3: Install Dependencies
    Run the following command to install the project dependencies:
    npm install

  • Step 4: Run
    Run the following command to run the project in development mode. It will start the development server.
    npm run dev
    Run the following command to build project.
    npm run build
    If you want to test your build project run the following command. It will run preview of your buit project.
    npm run start

Visit http://localhost:3000 in your browser to view your Fadmin Admin Template.

  • Customization
    • Fadmin-Next is highly customizable. Explore the project structure and configuration files to tailor the template to your specific needs. Key configuration files include:
  • next.config.js:
    • Customize Builds and next configuration as needed.
  • .eslintrc.json:
    • Customize ESlint rules for linting configurations.
    • For More Customization use different files mention bellow in folder struction. please visit that section
