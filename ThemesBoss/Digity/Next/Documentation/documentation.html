Installation

To setup the landing template, follow below-mentioned steps

  • Choose Next-friendly IDEs like Visual Studio Code for efficient development.
  • Download and install Node and npm from the official website https://nodejs.org/
  • Use npm install or npm install --save --legacy-peer-deps Command To install all node modules.
  • Use npm run dev command to run Next application.
  • To deploy the Next application, navigate to the project directory and execute the npm run build command to create a production build.
