Introduction

Kerri is a one-page responsive portfolio layout which can be utilized to demonstrate your inventiveness. It is produced utilizing the bootstrap and has taste of material plan. It's an ideal decision for a planner, engineer, specialist or some other expert who need to speak to his abilities and inventive work. The code is spotless and straightforward and enables any engineer to effectively alter it. You are important for us and so please feel free to get back to me with any question or feedback.

Installation

To setup the landing template, follow below-mentioned steps

  • Choose React-friendly IDEs like Visual Studio Code for efficient development.
  • Download and install Node and npm from the official website https://nodejs.org/
  • Use npm install or npm install --save --legacy-peer-deps Command To install all node modules.
  • Use npm start command to run react application.

Layout.js Structure 

import React from "react";
import Footer from "./helperComponents/Footer";
import Navbar from "./helperComponents/Navbar";
import { Outlet } from "react-router-dom";

import AllContent from "./helperComponents/AllContent";
import Loader from "./helperComponents/Loader";

function Layout() {
  return (
    <>
      {/*  start Loader */}
      <Loader />
      {/* End loader */}
      {/* start Navbar */}
      <Navbar/>
      {/* end navbar  */}

      {/* Start outlet */}
      <Outlet />
      {/* End of the outlet */}

      {/* all other content */}
      <AllContent />
      {/* content end*/}

      {/* start Fotter */}
      <Footer></Footer>
      {/* End Footer */}
    </>
  );
}

export default Layout;
