Introduction

Start with Falcon's multi-purpose template. Launch your campaign and take advantage of our expertise in designing and managing conversion-oriented Bootstrap 5 HTML pages.

Falcon is a powerful Saas and software starter template. This is a great HTML template for startups, co-working spaces, cloud storage, auto journeys, classic saas, classic apps, events, business, apps, education courses, personal portfolios, services, businesses, unique minimal portfolio products, saas, social media marketing, agency, client, marketing, digital courses, page Landing, etc. Falcon is fully updated with the latest Bootstrap v5.2.3 and Gulp files. We provide Dark version supported in Light and Dark versions.

Basic Setup for the Falcon HTML version template

Summary

Please follow below steps to install and setup all prerequisites:

  • 1.
    Nodejs

    Make sure Node.js is installed and running on your computer. If you already have Node installed on your computer, you can skip this step if your current Node version is higher than 14.

  • 2.
    Yarn

    Make sure Yarn is installed and running on your computer. If you already have Yarn installed on your computer, you can skip this step. We recommend using Yarn instead of NPM.

  • 3.
    Gulp

    Make sure that Gulp is installed and running on your computer. If you have gulp installed, run the command npm install -g gulp from your terminal.

  • Now, let's start the informative commands to run the application

    Before running below command make sure you have follow above 3 steps.

After you finished with the above steps, you can run the following commands into the terminal / command prompt from the root directory of the project to run the project locally or build for production use:

  1. Go to root of the project

  2. Go to html-gulp

  3. After that follow the below commands

Command Description
npm install This would install all the required dependencies in the node_modules folder.
gulp Runs the project locally, starts the development server and watches for any changes in your code, including your HTML, javascript, sass, etc. The development server is accessible at http://localhost:3000.
gulp build Generates a /dist directory with all the production files.
Dark Version
Dark Version :

In order to have Dark mode enabled, replace the reference of bootstrap.min.css to bootstrap-dark.min.css and also change style.css to style-dark.css in the src/partials/head-css.html

Quick customization of layout using scss.

You can make changes in the src/assets/scss/config/default/_variables.scss file.

Variables Description 
$primary:       $indigo;
$secondary:     $blue;
$success:       $green;
$info:          $cyan;
$warning:       $yellow;
$danger:        $red;
$light:         $gray-800;
$dark:          $gray-600;
 You can manage those colors to update the color theme.
